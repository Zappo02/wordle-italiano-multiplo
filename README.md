# 🇮🇹 Wordle Italiano

Indovina la parola italiana del giorno in 6 tentativi.

## Stack
- React 18 + Vite 5
- Singolo file `src/App.jsx`
- Deploy su Vercel via GitHub

## Sviluppo locale

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy su Vercel

1. Push su GitHub
2. Importa il repo su [vercel.com](https://vercel.com)
3. Framework preset: **Vite** — nessuna configurazione extra necessaria
